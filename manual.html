1このツールでできること

土木・建築現場の朝礼ネタを 155 件以上収録した、無料の Web アプリです。インストール不要、ブラウザを開くだけ。スマホ・タブレット・PC、どの端末からでも同じように使えます。

主な機能

  • 📅 今日のおすすめが時期・季節・曜日に合わせて自動表示
  • 🔍 作業内容／天候／対象者／雰囲気の4軸タグでしぼり込み
  • 🔊 音声読み上げで朝礼台本がそのまま読める
  • ⭐ お気に入り保存／使った記録／👍ウケた／📝役に立った を1タップ
  • 💬 公式LINE 毎朝7:00 自動配信＋キーワード送信で関連ネタ返信
  • 📱 看板用 QRコード 自動生成、誰でもスマホで開ける
  • 💾 マイデータバックアップで他端末にデータ移動可能

💴 完全無料

ログイン・アカウント作成・課金、全て不要です。URLにアクセスするだけ。

3🏠 ホーム ── 今日のおすすめを開く

アプリを開くと一番最初に表示される画面。朝、現場に着いたらまずここを開いてください。

表示の構成（上から）

  1. 今日の自動レコメンド ─ 月／曜日／月末などから自動選出した「一番おすすめのネタ」。この台本をひらくで本文が開きます。
  2. 今日のメイン作業は？ ─ 運転・クレーン・高所・足場・コンクリ・鉄筋・溶接・塗装・解体・電気・玉掛・全作業の12カテゴリボタン。タップすると「さがす」タブにジャンプして自動でしぼり込まれます。
  3. 今の時期のおすすめ ─ 今月該当のネタが下に続きます。下にスワイプして眺めてください。

💡 一番ラクな使い方

「今日の自動レコメンド」のこの台本をひらくを押すだけで、朝礼で話す内容が用意できます。読み上げ機能（🔊音声）と組み合わせれば、考える時間ゼロ。

6ネタを開いた画面（モーダル）の使い方

どこかのカードをタップすると、画面下から本文が出てきます（モーダル）。

上部

  • A− / A+：この画面だけ文字サイズを一時調整。朝礼で読み上げ中に便利。
  • ×：閉じる。

本文中の強調表示

  • 数値 は赤色（例：20kg、3分）
  • 行動 は黄色（例：必ず2人で持つ、指差し確認）

👍ウケた／📝役に立った ボタン

本文の下に2つの評価ボタンがあります。1ネタ1端末1回。押すと黄色く反転して「✓取消」が表示されます。もう一度押すとカウントを取消できます。

※ 端末ごとの記録です。複数の端末で別々にカウントされます。
複数端末で揃えたい時は「マイデータ バックアップ・復元」で同期できます（§8参照）。

8💾 マイデータのバックアップ・復元

各端末に保存されている「お気に入り・評価・使用履歴」を JSON ファイルで保存／復元できます。別の端末でも同じデータを使いたい時、またはツールがアップデートされた後も自分の履歴を残したい時に使います。

バックアップ手順（A端末で）

  1. ⚙️ 設定タブ → 「💾 マイデータのバックアップ・復元」
  2. ⬇️ マイデータをバックアップ をタップ
  3. chorei-mydata-YYYY-MM-DD-HH-MM-SS.json という名前のファイルがダウンロードされます
  4. そのファイルを AirDrop / iCloud Drive / メール / LINE などでB端末に送る

復元手順（B端末で）

  1. 受け取った JSON ファイルを B端末に保存
  2. ⚙️ 設定 → 「💾 マイデータのバックアップ・復元」
  3. ⬆️ マイデータを復元 をタップ
  4. JSON ファイルを選択
  5. 確認ダイアログで OK＝マージ または キャンセル＝全置換 を選ぶ
    • マージ（推奨）：既存データに上書きする。両方の端末で評価したネタは合算ではなく上書き。
    • 全置換：B端末の既存データを消して、JSONの内容だけにする。

✅ 安心ポイント

ツールがアップデートされても、この機能で自分のデータは保護できます。マイデータは端末に保存されているので消えません。

8.5📊 現場で使える無料テンプレート

設定画面に「📊 歩掛り記録テンプレート」のダウンロードボタンを用意しました。土木工事の歩掛りを自分で蓄積するためのGoogleスプレッドシートです。

テンプレートでできること

  • 毎日の作業実績（出来高・人工・時間など）を1行ずつ入力
  • 工種ごとの実績歩掛り（人工/単位、単位/人工、単位/日）が自動集計
  • 月次サマリで稼働日数・記録件数・総人工・平均気温を自動表示
  • 複数日にまたがる作業は「期間作業入力」シートで1行管理
  • マスタ（工種・職種・場所・規格など）はプルダウンから選択可能

使い方

  1. ⚙️ 設定タブ → 「📊 現場で使える無料テンプレート」
  2. 📊 歩掛り記録テンプレートを取得 をタップ
  3. Googleにログインしていれば、自動で「コピーを作成」ダイアログが出ます
  4. 「コピーを作成」ボタンを押すと、あなたのGoogleドライブに自分専用のコピーが保存されます
  5. 以降は自分のコピーを編集・蓄積していくだけ。元のテンプレートに影響することはありません

💡 みんなで1つを共有するのではなく、1人1コピー

テンプレートを取得すると、それぞれが自分専用のスプレッドシートを持つ形になります。他人の入力が混ざることはなく、自分の現場の実績歩掛りが自分の手元で育っていきます。

10朝礼看板に貼る（QR）

現場の朝礼看板や休憩所にQRコードを貼っておけば、職人さん全員が自分のスマホでツールを開けます。

  1. ⚙️ 設定タブ → 「朝礼看板用 QRコード」
  2. 表示されているQRを長押し → 画像保存（または右クリック保存）
  3. A4サイズくらいに印刷
  4. 看板や休憩所、ヘルメット入れの近くに掲示

💡 印刷時のコツ

「QR：現場朝礼ネタ・無料・スマホで読み取ってね」など説明文を添えると食いつきが良くなります。

11困ったとき・よくある質問

Q. 音声読み上げが動かない

A. ブラウザの設定で音声がOFFになっているか、その端末が日本語TTS非対応の可能性。Safari / Chrome の最新版で改善することが多いです。iPhoneは「設定 → アクセシビリティ → 読み上げコンテンツ」で日本語の音声が入っているか確認してください。

Q. 「今日のおすすめ」が同じネタばかり出る

2026年7月のアップデートで、当日が何日目かで自動ローテーションするようになりました。同じ日は同じネタが出ますが、翌日は違うネタになります。それでも見飽きたら「今の時期のおすすめ」から選ぶか、タグ検索を使ってください。

Q. 別の端末でも同じお気に入りを使いたい

A. §8 のバックアップ/復元機能を使ってください。

Q. LINE Botからキーワード送ったのに返信が来ない

A. 1～2分待っても返信がなければ、もう一度送り直してください。一時的に応答不可な場合があります。それでも届かない時は本ツール内の「💬LINE公式 友だち追加」ボタンからチャットでお問い合わせください。

Q. 看板用QRが小さくて読みづらい

A. 設定画面のQR画像を長押し→画像保存して、大きく印刷してください。A4サイズ全面に拡大すれば3mくらい離れていても読めます。

Q. ツールが急に動かなくなった

A. ブラウザのキャッシュをクリア（Cmd+Shift+R / Ctrl+Shift+R）してから再アクセスしてください。それでもダメな場合は1～2時間後に再試行。サーバ側の一時的な問題の可能性があります。
